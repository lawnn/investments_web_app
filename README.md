# 積み立て投資シミュレーション

このアプリはNASDAQ100,S&P500といったものを積み立てシミュレーションした場合の投資額をシミュレーションすることができます

# symbol
symbolはyfinanceに遵守したものとなっています

NASDAQ100は`^NDX`

S&P500は`^GSPC`

といったものです

# Monthly Investment

毎月の投資額(円)です

# Start Date

投資を始める時間です

# End Date

終わる時間です
